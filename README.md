# DailyIncomeLogger

โปรแกรมบันทึกรายรับประจำวัน พัฒนาโดยใช้ C# Windows Forms

## เทคโนโลยีที่ใช้
- C# Windows Forms
- .NET Framework
- Visual Studio 2022

## วิธีใช้งานโปรแกรม
1. กรอกรหัสรายการ (IncomeID)
2. กรอกชื่อรายการ (IncomeName)
3. เลือกหมวดหมู่ (Category)
4. เลือกวิธีรับเงิน (PaymentMethod)
5. กดปุ่ม "เพิ่มรายการ"

## การตรวจสอบข้อมูล
โปรแกรมจะตรวจสอบว่า:
- IncomeID และ IncomeName ต้องไม่ว่าง
- หากข้อมูลไม่ครบ จะแสดง MessageBox แจ้งเตือน

## ตัวอย่างการทำงาน
เมื่อกรอกข้อมูลครบ และกดปุ่มเพิ่มรายการ
จะแสดง MessageBox สรุปข้อมูลที่กรอก
